# Archive

This directory stores historical or reference-only material.

## Rules

- Archive material is not source of truth.
- If a file is needed for active design or implementation, move its conclusions into Markdown under `docs/`.
- Binary source material may be kept locally here for reference, but should not compete with active repo docs.
